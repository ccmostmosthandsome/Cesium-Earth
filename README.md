# Cesium-Earth
Use  CesiumJS to Imitate a "Google Earth"

- [x] 3D globe
- [x] layer chooser
- [x] map status
- [x] compass & my location
- [x] measure distance
- [ ] photos in globe (doing)
- [ ] hawkeye viewer (doing)
- [ ] explore
